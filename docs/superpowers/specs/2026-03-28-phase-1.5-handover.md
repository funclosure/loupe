### Gotchas (learned the hard way)
- **Milkdown injects CSS classes** (`prose`, `milkdown-theme-nord`) — stripped via MutationObserver in `milkdown-setup.ts`. Any new ProseMirror node views need to account for this.
- **Service worker is disabled** (`selfDestroying: true` in vite.config.ts) — re-enable for production later.
- **Server sends no-cache headers** on all static files — intentional during dev.
- **`@shared/*` path alias** must be configured in tsconfig.json, vite.config.ts, AND vitest.config.ts.
- **Agent SDK pattern**: `query()` + `MessageChannel` for multi-turn. Manual `.next()` on iterator, NOT `for await...of`. See `src/server/lens-session.ts`.
